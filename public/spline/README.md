# Spline Asset Setup

The file `living-blueprint.splinecode` in this directory is currently a **PLACEHOLDER**.

To complete the visual setup:

1.  Go to [Glass Knot Vortex](https://community.spline.design/file/90d98893-bcd5-4c28-8338-d322ff5c3655) on Spline Community.
2.  Click **Remix**.
3.  Implement the structure:
    - Root Group: `Blueprint`
    - Subgroups: `phase_0` to `phase_7`
4.  Add Variables:
    - `phase` (Number)
    - `progress` (Number)
5.  Set up logic:
    - Link `phase` variable to show/hide the subgroups.
6.  **Export**:
    - Click **Export**.
    - Select **Code**.
    - Choose **Vanilla JS** or **React** (doesn't matter, we need the file).
    - Wait for the URL to generate.
    - Click the **Download** icon (arrow down) in the code snippet area or next to the URL.
    - Save the file as `living-blueprint.splinecode`.
7.  Replace the file in this directory with your exported file.
