## Purpose

This index helps the Vantus team and approved stakeholders navigate the Vantus Care SOP library.
It is a controlled internal guide to the 31 numbered Care SOPs and the operating areas they support.

## What this index is for

Use this index to:

- find the right SOP for a service situation,
- understand which operating area owns the SOP,
- connect the SOP library to sales, onboarding, delivery, security, reporting, and offboarding.

## Key operating rules

- Care follows project delivery; it does not replace project scoping.
- Care uses the active tiers `Foundation`, `Advanced`, and `Sovereign`.
- Pricing, add-ons, and commercial commitments must align to the current `care/pricing/` and `care/commercial/` documents.
- Public claims belong in approved public documents, not in internal operating references.
