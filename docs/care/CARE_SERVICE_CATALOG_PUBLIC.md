## What Vantus Care is

Vantus Care is the recurring support layer that follows project delivery.
It is designed for clients who want ownership, visibility, and a predictable operating rhythm after launch.

## What makes Care different

- You keep ownership of accounts, hardware, code, and data.
- Pricing is transparent and modular.
- Security and documentation are part of the operating baseline.
- Support is structured around proof, reporting, and clear escalation.

## Care tiers

### Foundation

Best for smaller environments that need dependable day-to-day support.

Includes:

- help desk coordination,
- monitoring and alert routing,
- patch and backup oversight,
- baseline reporting,
- recurring operating review.

### Advanced

Best for growing businesses that need stronger security and more management depth.

Includes everything in Foundation, plus:

- stronger reporting cadence,
- security operations support,
- compliance-support coordination,
- higher-priority response handling,
- more structured planning.

### Sovereign

Best for organizations that need the highest governance depth, strategic support, and specialized escalation.

Includes everything in Advanced, plus:

- deeper quarterly planning,
- expanded stakeholder governance,
- strategic risk and lifecycle planning,
- specialized support options,
- higher-touch operating rhythm.

## Common add-ons

- server management
- application server management
- database server management
- after-hours coverage
- project work outside recurring scope
- hardware procurement assistance
- compliance evidence support

See `pricing/CARE_ADDON_MATRIX.md` for the current add-on structure.

## What Care does not include by default

- major project work without written scope
- custom software builds under recurring support
- hardware leasing or bundled device financing
- legal, audit, or certification guarantees

## Service boundary rule

If the work needs new scope, significant change, or non-routine delivery effort, it moves into a scoped project or change request.

## Best-fit clients

Care is best for clients who:

- already have a Vantus-delivered system, or a system that can be stabilized,
- want ownership and exit-readiness,
- value transparent operations over vague retainers,
- need a disciplined support layer without MSP lock-in.
