## How to Maintain This Inventory

1. **Update monthly**—add new assets, remove old ones
2. **Review quarterly**—verify accuracy
3. **Check annually**—full audit
4. **Before purchases**—check what you already have
5. **For insurance**—document value

## Best Practices

- Assign an owner to each asset
- Label physical assets with asset tags
- Track software licenses
- Monitor warranty expirations
- Review lifecycle status regularly
- Document dependencies
