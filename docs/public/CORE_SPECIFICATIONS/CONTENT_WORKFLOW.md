# Public Content Workflow

Version: 2.0
Last updated: 2026-03-06

## Workflow

Draft -> Review -> Approved -> Published

## Public content rules

- No publishing without review approval.
- Case studies must include evidence sections.
- Pricing statements must match `../../pricing/pricing_public.yaml` exactly.
- Copy must follow public-safe language and avoid internal controls detail.
- Offer framing must match `../../enterprise_pack/03_offers/SERVICE_CATALOG_PUBLIC.md`.

## Minimum role model (public-safe)

- Editor: draft and update content.
- Reviewer: validate claims and route fit.
- Publisher: approve and publish.

For CMS implementation details, follow `../../enterprise_pack/05_architecture/ADR-0001-CMS-Choice.md`.
