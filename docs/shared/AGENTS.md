# Shared Documentation - AGENTS

**Version:** 1.0.0
**Last updated:** 2026-03-08
**Applies to:** Shared documentation and cross-surface governance docs

## Purpose

This file defines how AI agents may work on `shared/` documentation.

The goal is to keep cross-cutting standards accurate, calm, and reusable across the full Vantus documentation tree.

## Allowed work

- update shared governance, security, style, and operational standards
- reconcile terminology across surfaces
- fix broken cross-surface references
- document cross-cutting rules that apply to more than one product surface

## Disallowed work

- do not turn `shared/` into a portal-only or admin-only spec set
- do not add pricing promises or product claims without canonical support
- do not duplicate surface-specific PRDs or route maps here
- do not invent compliance, performance, or security outcomes without evidence

## Review bar

Every change in `shared/` should preserve:

- plain language
- evidence-based claims
- surface-neutral wording unless a file explicitly says otherwise
- consistency with canonical pricing and offer docs
