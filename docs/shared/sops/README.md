# Vantus Systems Standard Operating Procedures (SOPs)

This directory contains the Standard Operating Procedures (SOPs) for Vantus Systems. SOPs are detailed, written instructions that describe how to perform a routine activity. They are a critical component of our quality system and ensure that we operate in a consistent and compliant manner.

## How to Use These SOPs

- **`SOP-SYSTEM-INDEX.md`**: This is the master document for all SOPs. It provides a summary of all SOPs, their status, and their owners. Start here to find the SOP you are looking for.
- **`SOP-000-SOP-Framework.md`**: This document explains the framework for our SOP system. It describes how SOPs are created, reviewed, and updated.

## SOP Domains

Our SOPs are organized into the following domains:

- **Commercial & Engagement (VS-BUS-XXX)**
- **Operational & Incident Management (VS-OPS-XXX)**
- **Engineering & Code Governance (VS-ENG-XXX)**
- **Security & Compliance (VS-SEC-XXX)**
- **People & Culture (VS-PEO-XXX)**

Each domain has a set of SOPs that are specific to that area.

## Contributing to the SOPs

Our SOPs are living documents and we welcome contributions from everyone. If you have a suggestion for how to improve an SOP, please file an issue in this directory.
