# SOP-CL-007: Ongoing Service Delivery

## 1. PURPOSE
To define the day-to-day operational model for delivering Vantus managed services after onboarding.

## 2. SCOPE
Applies to Help Desk, NOC/SOC teams, Delivery Managers, and Account Managers.

## 3. RESPONSIBILITIES
| Role | Responsibility |
|------|----------------|
| Help Desk | First-line support and ticket triage |
| NOC/SOC | Monitoring, incident response, and threat detection |
| Delivery Manager | SLA monitoring and escalations |

## 4. PROCEDURE
### Daily Operations
- Monitor alerts, respond to tickets per SLA, perform scheduled maintenance.

### Weekly Tasks
- Review backups, run health checks, and perform minor updates.

### Monthly Tasks
- Run patching cycles, security scans, and review system capacity.

## 5. TRIGGERS
- New incident, monitoring alert, scheduled maintenance window.

## 6. ESCALATION
- Follow escalation matrix in SOP-CL-008 and company escalation procedures.

## 7. QUALITY CRITERIA
- SLA compliance >95% monthly; CSAT >4.5/5 on ticket surveys.
