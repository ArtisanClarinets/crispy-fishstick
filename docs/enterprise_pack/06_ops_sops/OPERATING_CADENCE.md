# Operating Cadence (Solo Founder → Small Team)
**Version:** 1.0
**Date:** 2026-03-05

## Weekly rhythm
### Monday (60 minutes)
- Review pipeline
- Set weekly targets (Reality Checks, calls, build milestones)
- Identify the one constraint this week

### Daily (15 minutes)
- Update VSP-lite
- Pick top 3 tasks

### Friday (45 minutes)
- Review what shipped
- Capture one proof artifact
- Update backlog

## Metrics you review weekly
- Reality Checks produced
- Calls booked
- Deals closed
- Delivery cycle time
- QA gate pass rate

## Rule
If you don’t track it weekly, you don’t actually care about it.
