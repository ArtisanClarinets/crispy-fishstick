# Risk Register (Top 10)
**Version:** 1.0
**Date:** 2026-03-05

1) Scope creep → enforce change control.
2) Content delays → “no content, no launch” gate.
3) Overbuilding VSP → ship VSP-lite first.
4) Low lead flow → Reality Check outbound cadence.
5) Underpricing → minimum project margin rules.
6) One-off custom features → force module promotion.
7) Client ownership confusion → ownership checklist at kickoff.
8) Hosting outages → monitoring + backups + runbook.
9) Security incident → incident response SOP.
10) Burnout → weekly operating cadence and boundaries.
