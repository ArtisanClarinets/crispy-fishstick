# Metrics Dashboard (Vantus)
**Version:** 1.0
**Date:** 2026-03-05

## North Star metric
**Qualified leads generated per client site per month.**

If you can’t improve this, your work is “pretty” not “valuable.”

## Leading indicators
- Reality Checks delivered/week
- Calls booked/week
- Proposals sent/week
- Delivery cycle time
- QA gate pass rate (first pass)

## Lagging indicators
- Close rate
- Avg project value
- Referral rate
- Review velocity (Google reviews/month)

## Targets (first 90 days)
- 10 Reality Checks/week
- 3 calls/week
- 1–2 closes/month
- 2 proof pages/month
- 10 five-star reviews by day 90
