# Pricing Ranges (Internal) — Enterprise Delivery Track

> **Strategy:** Option A — Gulf Coast Leader
> Revenue Target: $500K Y1 → $1.4M Y2 → $2.2M Y3

**Version:** 2.1
**Date:** 2026-03-15

These are **starting ranges** to prevent underpricing and protect premium positioning.
Final pricing depends on complexity scoring, integration count, and written scope.

## Package 1: Website Rebuild (Modern + Complete)

- Typical: **$15,000 – $55,000**
- Use the high end when: >20 pages, high-stakes SEO migration, advanced UX flows, or multilingual content.

## Package 2: Website + CMS (Recommended)

- Typical: **$32,000 – $125,000**
- Use the high end when: multi-role approvals, migration complexity, multiple content types, or advanced editorial governance.

## Package 3: Website + Business Portal

- Typical: **$90,000 – $400,000**
- Use the high end when: RBAC + audit controls + workflow automation + system integrations are all in scope.

## Infrastructure

- Cloud deployment baseline: **$7,000 – $30,000**
- On-prem install + hardening: **$15,000 – $70,000** (hardware excluded)

## Standard add-on ranges

- Discovery + solution blueprint: **$5,000 – $20,000**
- Data/content migration: **$4,000 – $30,000**
- Integration (per system): **$8,000 – $35,000**
- SSO + RBAC hardening: **$12,000 – $50,000**
- Analytics/dashboarding: **$7,000 – $30,000**
- Localization framework: **$8,000 – $40,000**

## Margin and discount guardrails

- Gross margin floor: **50% minimum** (`../../pricing/pricing_ops.yaml`)
- Discount without founder approval: **max 5%**
- Discount with founder approval: **max 15%**

## Rule

If the quoted scope cannot hold margin, split into phases or remove scope.
