# Roadmap — 12 to 24 Months (Gulf Coast Leader)

> **Strategy:** Option A — Gulf Coast Leader
> Revenue Target: $500K Y1 → $1.4M Y2 → $2.2M Y3

**Date:** 2026-03-15

## Guiding constraint

You are bootstrapped. That means you must optimize for:

- cash collection,
- cycle time,
- repeatability,
- proof assets.

If a roadmap item doesn’t improve one of those, it is probably ego.

---

## Month 0–3: Prove the offer + build proof assets

**Goal:** 3–5 paid clients shipped with case studies.

**Ship:**

- Productized website + CMS package.
- SOP-driven delivery.
- Proof artifacts template pack.

**Build (internal):**

- VSP-lite: pipeline → project → deliverables → handoff tracker.

**Exit criteria:**

- 3 signed case studies,
- 2 referrals,
- delivery cycle time baseline measured.

---

## Month 3–6: Standardize delivery + reduce cycle time

**Goal:** 10 total projects shipped; cycle time down 20%.

**Build:**

- Vantus Seed baseline repo (reusable starter):
  - CMS integration,
  - SEO + accessibility scaffolding,
  - deployment templates,
  - proof artifact generator (Lighthouse/a11y/security headers).

**Sales:**

- Install local “authority flywheel”: reviews, before/after, local pages.

---

## Month 6–12: Expand into workflows + portals (modular)

**Goal:** 20–30 total projects shipped; start $25k–$75k MRR in support/maintenance add-ons (optional).

**Build:**

- VSP module 1: content workflow + approvals.
- VSP module 2: deployment + environment status.
- VSP module 3: change requests + audit trail.

**Offer expansion:**

- add “Customer Portal” module for verticals that need it.
- add “Admin Portal” module for internal workflows.

---

## Month 12–24: Become the default premium local choice

**Goal:** 50+ shipped projects, strong referral engine, repeatable portal modules.

**Build:**

- packaged vertical templates (without becoming templated trash):
  - dental/medical,
  - legal,
  - trades.

**Scale lever:**

- hire only when SOPs and VSP reduce the marginal cost of each new project.

---

## Red-team questions (answer honestly)

1. What stops you from spending 6 months building VSP and selling nothing?
2. What’s your minimum acceptable project margin?
3. Which 3 verticals will you refuse to serve (because they waste time)?
4. What is your “no content, no build” enforcement mechanism?
