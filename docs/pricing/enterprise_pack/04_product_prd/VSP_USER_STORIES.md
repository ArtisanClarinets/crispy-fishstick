# VSP User Stories (MVP)
**Version:** 1.0
**Date:** 2026-03-05

## CRM-lite
1. As the founder, I can create a lead with stage and next action so nothing drops.
   - Acceptance: lead list sortable by next_action_due.

2. As the founder, I can attach a Reality Check PDF to a lead.
   - Acceptance: file stored and visible on lead record.

## Delivery tracker
3. As the founder, I can create a project with package type and target launch date.
   - Acceptance: project appears on dashboard with status.

4. As the delivery owner, I can see a checklist of required deliverables.
   - Acceptance: cannot mark project “deploy-ready” until required deliverables are complete.

## Content workflow
5. As the client, I can approve a page’s content.
   - Acceptance: approval logged with timestamp.

## Change control
6. As the client, I can submit a change request.
   - Acceptance: request shows “submitted” and notifies owner.

7. As the founder, I can estimate impact and require approval.
   - Acceptance: CR cannot move to “in_progress” without approval.
