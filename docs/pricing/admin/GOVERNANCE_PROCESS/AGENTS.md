# Admin Documentation - AGENTS

**Version:** 1.0.0
**Last updated:** 2026-03-08
**Applies to:** Admin documentation and admin platform planning

## Purpose

This file defines how AI agents may assist with the admin platform documentation set.

## Allowed work

- draft admin requirements, checklists, and implementation notes
- review RBAC, audit logging, pricing governance, and internal operations flows
- improve admin runbooks, release process notes, and support templates

## Disallowed work

- do not treat admin docs as portal docs
- do not move portal or public requirements into the admin scope
- do not publish pricing, compliance, or security claims without canonical support

## Review bar

Admin changes should preserve:

- least-privilege and auditability
- pricing integrity and no hardcoded commercial logic
- plain language for internal operators
- traceable links back to canonical admin and shared documents
