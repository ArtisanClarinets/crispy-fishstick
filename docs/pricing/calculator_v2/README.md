# Vantus Systems Package Pricing Calculator v2

## Overview

This interactive pricing calculator allows clients to explore Vantus Systems build service packages, customize their solution, and generate accurate budget ranges in real-time.

- **No installation required** - just open in a web browser and and get started
- **Professional design** aligned with Vantus branding

## Quick Start

1. Open `index.html` in your web browser
2. Select a package tier
3. Adjust complexity factors (pages, integrations, functionality, migration, compliance, timeline pressure)
4. Check/uncheck add-on modules
5. Review your real-time estimate
6. Download or print estimates for client discussions

- **Fully responsive** - works on all screen sizes

- **Professional UI** with Vantus branding
- **Calculator logic** provides range-based pricing based on complexity factors and add-on modules

- **Professional disclaimers** included

- \*\*References pricing_public.yaml for accurate pricing data
