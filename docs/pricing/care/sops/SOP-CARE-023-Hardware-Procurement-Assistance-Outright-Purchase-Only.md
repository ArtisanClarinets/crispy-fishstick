## Step-by-Step: Hardware Procurement

### Step 1: Confirm Scope

Define engagement:
| Option | Description |
|--------|-------------|
| **Advisory Only** | We recommend, you buy direct |
| **One-Time Resale** | We source, you pay invoice |
| **No Financing** | Monthly payments never offered |

### Step 2: Draft Options

We provide:
| Level | Description |
|-------|-------------|
| **Good** | Meets minimum requirements |
| **Better** | Improved performance/features |
| **Best** | Maximum capability |

Each option includes:

- Specifications
- Tradeoffs
- Warranty assumptions
- Operating cost estimates

### Step 3: Separate Implementation

If setup needed:

- Implementation = separate project
- Scoped separately from Care
- One-time project fee

### Step 4: Purchase Outright

Payment options:

- Direct purchase by client
- One-time invoice from Vantus
- **Never**: Monthly payments or leases

### Step 5: Record Assets

Update:

- Inventory (SOP-CARE-010)
- Warranty register (SOP-CARE-024)
- Documentation

### Step 6: Decline Leasing

If leasing requested:

- Restate policy
- Provide outright options
- Document request and response

## Real Example

**Scenario**: Client needs a new server.

**Procurement Process**:

1. Assess requirements:
   - Users: 25
   - Applications: Database, file sharing
   - Growth: 20% per year

2. Provide three options:
   - Good: $3,500 (meets current needs)
   - Better: $5,200 (room to grow)
   - Best: $7,800 (maximum performance)

3. Explain operating costs (power, cooling, maintenance)

4. Client selects Better option

5. One-time invoice issued

6. Equipment delivered and installed

7. Asset recorded in inventory

8. Warranty tracked

## Key Terms Explained
