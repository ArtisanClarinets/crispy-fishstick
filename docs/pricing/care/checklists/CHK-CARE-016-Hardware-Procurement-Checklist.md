# CHK-CARE-016: Hardware Procurement Checklist

**Document ID:** VS-CARE-CHK-016  
**Version:** 1.0.0  
**Effective Date:** 2026-02-21  
**Owner:** VP of Delivery  
**Status:** Active  
**Reading Level:** 8th Grade  
**Estimated Time:** 2-4 hours (research + quote + procurement)  
**Risk Rating:** Medium (wrong hardware = wasted budget, poor performance)

**Goal:** Guide clients to appropriate hardware purchases with clear options, transparent pricing, and documented ownership—never leasing, always outright purchase.

---

## 1. Requirements Gathering

- [ ] Business need confirmed (why is hardware needed?)
- [ ] Current pain points documented (what's not working now?)
- [ ] User count determined (how many people affected?)
- [ ] Growth projection captured (expansion plans for 3-5 years)
- [ ] Application requirements listed (what software will run?)
- [ ] Performance needs defined (speed, storage, memory minimums)
- [ ] Special features noted (redundancy, mobility, specific ports)
- [ ] Budget range discussed (investment comfort zone)

**Time Estimate:** 30-45 minutes  
**Owner:** VP of Delivery / Engineer  
**CMMC Alignment:** SA.1.1 (supply chain), SA.2.1 (supply chain risk)

---

## 2. Research & Options Development

- [ ] Market research completed (current models, pricing)
- [ ] Vendor evaluation done (reputation, support quality)
- [ ] Three options prepared:
  - **Good:** Meets minimum requirements (budget-friendly)
  - **Better:** Improved performance/features (balanced)
  - **Best:** Maximum capability (future-proofed)
- [ ] Specifications documented for each option
- [ ] Tradeoffs clearly explained (what you gain/lose at each level)
- [ ] Warranty terms compared (length, coverage, support)
- [ ] Lead times verified (how long to receive)

**Time Estimate:** 45-90 minutes  
**Vantus standard:** always providing three options with clear tradeoffs

---

## 3. Total Cost of Ownership (TCO) Analysis

- [ ] Hardware cost itemized (purchase price only, no hidden fees)
- [ ] Operating costs estimated (power, cooling, space)
- [ ] Maintenance costs projected (warranty, support contracts)
- [ ] Lifecycle cost calculated (5-year total)
- [ ] Existing hardware trade-in value researched (if applicable)
- [ ] Financing options declined (per policy: outright purchase only)

**Time Estimate:** 20-30 minutes

---

## 4. Quote Preparation

- [ ] Bill of Materials (BOM) created (line-item list)
- [ ] Pricing current (quotes valid for 30 days noted)
- [ ] Shipping/delivery included (or noted as separate)
- [ ] Tax calculated (if applicable)
- [ ] Total clearly stated (single number for approval)
- [ ] Warranty details specified (what's covered, for how long)
- [ ] Delivery timeline confirmed (when it will arrive)

**Time Estimate:** 15-30 minutes

---

## 5. Client Presentation

- [ ] Options presented clearly (Good/Better/Best with visuals)
- [ ] Business value explained (not just specs, but benefits)
- [ ] Tradeoffs discussed openly (pros and cons of each)
- [ ] Questions answered thoroughly (technical and business)
- [ ] Recommendation provided (which option fits their needs)
- [ ] Timeline reviewed (when to order, when delivered)
- [ ] Next steps clarified (what happens after approval)

**Time Estimate:** 30-45 minutes

---

## 6. Approval & Procurement

- [ ] Client selection confirmed (which option chosen)
- [ ] Written approval received (email or signed quote)
- [ ] No leasing verified (payment terms: full purchase)
- [ ] Implementation needs assessed (separate project scope?)
- [ ] Order placed with vendor (purchase order issued)
- [ ] Tracking information obtained (shipment monitorable)
- [ ] Delivery appointment scheduled (coordinated with client)

**Time Estimate:** 20-30 minutes  
**Critical Policy:** Hardware is purchased outright—no leasing, no monthly hardware fees

---

## 7. Receipt & Verification

- [ ] Delivery received (inspected for damage)
- [ ] Contents verified (everything on BOM received)
- [ ] Serial numbers recorded (for warranty tracking)
- [ ] Warranty registration completed (with manufacturer)
- [ ] Physical condition documented (photos if concerns)
- [ ] RMA process noted (if anything needs return)

**Time Estimate:** 15-30 minutes

---

## 8. Asset Recording

- [ ] Asset added to inventory (TPL-CARE-010)
- [ ] Warranty dates recorded (start and end dates)
- [ ] Vendor support info documented (contact numbers, contracts)
- [ ] Location noted (where installed)
- [ ] Owner assigned (who manages this asset)
- [ ] Lifecycle tracking initiated (refresh date calculated)
- [ ] Documentation filed (receipts, warranties, manuals)

**Time Estimate:** 15-20 minutes  
**CMMC Alignment:** CM.2.1 (baseline configurations), CM.3.1 (change control)  
**Owner:** Delivery Lead

---

## 9. Implementation (if applicable)

- [ ] Separate project scoped (if installation complex)
- [ ] Project SOW created (implementation work documented)
- [ ] Schedule coordinated (minimal business disruption)
- [ ] Configuration completed (installed and tested)
- [ ] User training provided (if needed)
- [ ] Handoff documented (client accepts completed work)

**Time Estimate:** Varies by complexity

---

## Procurement Policy Reminders

| We Always Do | We Never Do |
|--------------|-------------|
| Outright purchase | Leasing or financing |
| Clear ownership transfer | Hardware subscriptions |
| Three options with tradeoffs | Single "take it or leave it" option |
| Transparent pricing | Hidden fees or bundled costs |
| Client makes final decision | Pressure tactics |

---

## Success Criteria

| Criterion | Target | Verification |
|-----------|--------|--------------|
| Client satisfaction | 95%+ | Post-procurement survey |
| On-budget delivery | 100% | Invoice vs. quote |
| On-time delivery | 95%+ | Delivery vs. promise |
| Zero leasing offered | 100% | Quote audit |
| Correct specifications | 100% | Receipt verification |
