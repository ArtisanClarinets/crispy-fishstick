# CARE_PRICING_PUBLIC - Vantus Care

**Audience:** Public, sales, delivery
**Last updated:** 2026-03-07
**Status:** Canonical public pricing baseline

## Pricing model

Vantus Care uses a per-user recurring model with minimum monthly commitments.
Final pricing still depends on scope, support boundaries, environment complexity, and approved add-ons.

## Starting prices

| Tier | Starting price | Minimum monthly |
|---|---:|---:|
| Foundation | $175 per user / month | $2,500 |
| Advanced | $225 per user / month | $5,000 |
| Sovereign | $300 per user / month | $10,000 |

## What drives price

- user count and environment size
- support window expectations
- reporting and governance cadence
- security and compliance-support depth
- server or specialized infrastructure coverage
- after-hours or specialized escalation requirements

## Common add-on starting prices

| Add-on | Foundation | Advanced | Sovereign |
|---|---:|---:|---:|
| Basic server management | $129 / month | $99 / month | $79 / month |
| Application server management | Quote as scoped | $99 / month | $79 / month |
| Database server management | Quote as scoped | Quote as scoped | $99 / month |

See `CARE_ADDON_MATRIX.md` for availability and packaging rules.

## Commercial notes

- Recurring Care does not include major project work.
- Hardware is purchased outright by the client; Vantus does not lease devices.
- Contract term, notice period, and after-hours coverage are defined in the applicable service order.
- If the environment is unusually complex, Vantus may quote outside the starting baseline.
