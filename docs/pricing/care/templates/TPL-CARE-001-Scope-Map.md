## Risk Factors

Potential issues and how we'll handle them:

| Risk | Likelihood | Impact | Risk Score | How We'll Mitigate | CMMC Ref |
|------|------------|--------|------------|-------------------|----------|

| | | | | | |

**Risk Scoring:** Severity (1-5) × Likelihood (1-5) = Score (1-25)
- 16-25: Critical - immediate action
- 9-15: High - address soon
- 4-8: Medium - monitor and plan
- 1-3: Low - accept or monitor
