## How to Use This Register

1. **Keep it current**—update when access changes
2. **Review regularly**—quarterly at minimum
3. **Verify accuracy**—confirm with account holders
4. **Remove old entries**—archive terminated access
5. **Use for audits**—evidence of access control

---

## Best Practices

- Review quarterly
- Remove access immediately when people leave
- Require MFA for all admin accounts
- Document all exceptions
- Use named accounts (not shared)
- Verify access annually with users
