## Scoring System

### Freshness Levels

| Status | Age | Score | Action Required |
|--------|-----|-------|-----------------|
| 🟢 **Current** | <30 days | 100% | None |
| 🟡 **Recent** | 30-60 days | 80% | Monitor |
| 🟠 **Aging** | 60-90 days | 60% | Schedule update |
| 🔴 **Stale** | >90 days | 40% | Update immediately |
| ⚫ **Obsolete** | Unknown | 0% | Create or archive |

## How to Improve Your Score

1. **Set calendar reminders** for document reviews
2. **Update immediately** when procedures change
3. **Assign clear owners** for each document
4. **Archive obsolete** docs (don't delete)
5. **Version control** all changes
6. **Review quarterly** at minimum
